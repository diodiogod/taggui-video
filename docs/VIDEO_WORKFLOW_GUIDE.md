# Video Workflow Guide

Quick guide for video-first usage.

## Typical Workflow

- Load folder with videos
- Use timeline and frame controls for navigation
- Set loop markers and playback speed
- Use spawned viewers for side-by-side checking

## Multi-Viewer Usage

- Spawn from View menu or context flow
- Keep one active controls owner at a time
- Use sync actions when comparing multiple videos
